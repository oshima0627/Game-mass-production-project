# CLAUDE.md

## プロジェクト概要

ゲーム量産プロジェクト。複数のゲームを効率的に開発・管理するためのプロジェクトです。

## ディレクトリ構成

```
Game-mass-production-project/
├── CLAUDE.md          # このファイル
├── games/             # 各ゲームのソースコード
├── shared/            # 共通ライブラリ・アセット
├── tools/             # 開発ツール・スクリプト
└── docs/              # ドキュメント
```

## 開発フロー

1. `games/` 配下に各ゲームのディレクトリを作成する
2. `shared/` の共通コンポーネントを活用する
3. 変更は `claude/` プレフィックスのブランチで開発し、PRを作成する

## よく使うコマンド

```bash
# 依存関係のインストール（プロジェクトに応じて変更）
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# テスト実行
npm test
```

## コーディング規約

- インデント: スペース2つ
- ファイル名: ケバブケース（例: `game-scene.ts`）
- クラス名: パスカルケース（例: `GameScene`）
- 変数・関数名: キャメルケース（例: `gameState`）

## 注意事項

- 共通アセットを変更する場合は、他ゲームへの影響を確認すること
- 新しいゲームを追加する際は、`docs/` にゲームの仕様書を作成すること
