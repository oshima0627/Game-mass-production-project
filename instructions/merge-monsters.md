# Merge Monsters 開発指示

## 最重要方針
- HTML/JS/CSS + Capacitor のプロジェクト構造で作る
- コードは最後まで書き切る。省略・TODO・「以下同様」は禁止
- スマホ縦持ち最優先、ストアに出せる品質にする
- 広告は @capacitor-community/admob で実装

## 概要
ポップでかわいいモンスター合体パズルゲーム。4×4グリッド上で同じレベルのモンスターをドラッグ＆ドロップで合体させ、より高レベルのモンスターに進化させる。進化したモンスターはコインを自動生成し、コインで新しいモンスターを召喚。グリッドが満杯になるとゲームオーバー。

## 技術スタック
- JavaScript (ES2020+) / 素のCanvas 2D / Capacitor 6 / @capacitor-community/admob
- プロジェクト構造: www/ 以下に index.html, css/style.css, js/(main.js, game.js, grid.js, monster.js, renderer.js, input.js, sound.js, particles.js, ui.js, ads.js)

## 操作
- ドラッグ＆ドロップのみ（片手操作）
- モンスターをタッチして同レベルのモンスターの上にドラッグで合体
- 空きマスにドラッグで移動（配置変更）
- 画面下部の「召喚」ボタンタップでランダムなLv.1〜2モンスターを空きマスに召喚

## ゲームルール
- 4×4（16マス）のグリッド
- 同レベルのモンスター同士を重ねると合体 → レベル+1のモンスターに進化
- モンスターはLv.1〜Lv.15まで進化可能
- 各モンスターはレベルに応じたコインを毎秒自動生成（Lv.1: 1コイン/秒、Lv.n: n^1.5コイン/秒）
- 召喚コスト: 基本10コイン、召喚回数に応じて緩やかに上昇（10 × 1.08^召喚回数）
- グリッドが満杯かつ合体可能なペアがない場合にゲームオーバー
- マイルストーン: Lv.5, Lv.8, Lv.10, Lv.12, Lv.15到達時にボーナス演出＋コインボーナス
- スコア = 合体した回数 × 最高到達レベル。ハイスコアはlocalStorageに保存

## モンスターデザイン（Canvas 2Dで描画）
レベルごとに色と表情が変化する丸いモンスター:
- Lv.1: 水色 / にっこり目
- Lv.2: 緑 / ウインク
- Lv.3: 黄色 / びっくり目
- Lv.4: オレンジ / ニヤリ
- Lv.5: 赤 / キラキラ目 + 王冠
- Lv.6: 紫 / ドヤ顔
- Lv.7: ピンク / ハート目
- Lv.8: 青 / サングラス
- Lv.9: 金色 / 炎エフェクト
- Lv.10〜15: レインボーグラデーション + 各種豪華装飾
- 全レベル共通: 丸い体、短い手足、レベル数字を体の中央に表示

## 難易度カーブ
- 序盤（〜Lv.3到達）: 召喚コスト低い、グリッドに余裕あり。合体の基本を学習
- 中盤（Lv.4〜7）: 召喚コスト上昇でグリッド管理が重要に。不要な低レベルの整理が鍵
- 終盤（Lv.8〜）: 空きマスの戦略的確保が必須。高レベル合体の達成感がモチベーション

## ビジュアル
- 背景: パステルグラデーション（薄い水色→薄いラベンダー）、ゆっくり色が変化
- グリッド: 白の角丸カード風、薄いシャドウ付き
- 空きマス: 点線の円（ここに置けるよガイド）
- テキスト: ダークグレー(#333)、アクセント: コーラルピンク(#FF6B6B)
- 全体的にパステル＆ポップ、老若男女に受ける明るいトーン

## ゲームフィール（必ず実装）
- 合体演出: 2体が吸い込まれ → 新モンスターがバウンスしながら登場 + パーティクル爆発
- レベルアップ演出: マイルストーン到達時に画面全体に紙吹雪 + 「NEW!」テキスト
- ドラッグ中: 持ち上げたモンスターが少し拡大 + 影が離れる（浮いている感）
- 合体可能ヒント: 同レベル同士が隣接すると軽く光るパルス
- コイン生成: モンスターから小さなコインアイコンがポップして上に消える
- サウンド: Web Audio APIで生成（合体音、召喚音、コイン音、マイルストーン音、ゲームオーバー音）
- NEW RECORD演出

## 画面構成
- **タイトル**: ゲームロゴ（ポップ体）、最高到達レベル、ハイスコア、STARTボタン
- **プレイ**: グリッド中央配置、HUD上部（コイン数 左・スコア 右）、召喚ボタン下部（大きく押しやすく）、バナー広告最下部60px
- **ゲームオーバー**: フェード暗転、スコア/最高レベル/ハイスコア表示、リワード広告「広告を見てグリッドを2マス空ける」、リスタートボタン
