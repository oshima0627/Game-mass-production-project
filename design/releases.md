# Releases 台帳

実装フェーズに入った作品の一覧。STEP 4 で実装リポジトリを作成したタイミングでエントリを追加し、ステータス遷移時に更新する。

## ステータスの取りうる値

| ステータス | 意味 |
|------|------|
| `design` | 設計フェーズ（本リポジトリの `design/<title>.md` で作業中） |
| `in-development` | 実装リポジトリ作成済み、開発中 |
| `soft-launched` | 特定国でソフトローンチ開始、指標検証中 |
| `global` | グローバルリリース済み |
| `maintenance` | 運営フェーズ（バグ修正・イベント運用中心） |
| `sunset` | サービス終了 |

## 更新タイミング

- `in-development`: 実装リポジトリ作成直後（STEP 4 完了時）
- `soft-launched`: フィリピン/カナダ等でのストア公開時
- `global`: 全地域公開時
- `maintenance`: アクティブ開発終了時
- `sunset`: サービス終了決定時

ステータス遷移のタイミングで作品固有の学びを `design/knowledge/` に抽出する。
